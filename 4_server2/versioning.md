   "express": "^5.2.1",

1st part -> 5
2nd part -> 2
3rd part -> 1

# 3rd part 
   |--> Minor fixes (optional updates)

# 2nd part 
   |--> Recommended bug fixes (security fix)

# 1st part 
   |--> Major / Breaking update

# ^ mean
   |--> install recommended and minor fixes automatically without changing 1st part

# ~ mean
   |-->  install only minor fixes automatically
